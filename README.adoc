= Back-template.

https://coveralls.io/github/Meat-Hook/gback-template?branch=master[image:https://coveralls.io/repos/github/Meat-Hook/back-template/badge.svg?branch=master[Coverage
Status]]

Репозиторий содержит шаблон для проектов, процент покрытия тестов высокий,
есть интеграционные тесты, которые запускаются с dockertest, чтобы затянуть инстанс базы.

Архитектура каждого модуля (микросервиса) является луковой, центр (app) содержит бизнес логику проекта,
не зависит никак от внешних слоев, так как все внешние зависимости (БД, API, GRPC) задаются через жесткие контракты
Остальные же слои, (repo, API, etc) являются максимально тонкими и не содержат никакой логики, исключительно выполнение
жестко заданных контрактов бизнес логики.

== Для старта нужно

* docker-compose up --build

Для остановки

* docker-compose down
